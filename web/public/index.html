My machine-learning forecast of the 2026 World Cup, built entirely from public data (Code & method on GitHub). Each team gets a strength rating from its national-team Elo (computed from past international results) and the market value of its squad (Transfermarkt). My model turns those into odds for every match, then plays the rest of the tournament out 100 000 times; the percentages below are how often each team reaches each stage. (I also rate players from club stats — FBref and Understat — but it didn't improve the forecast, so the live model leaves it out.)

Model updates itself: after every match the new result feeds back in, the model re-runs, and this page refreshes — so the forecast always reflects what's happened so far.

Betting odds are never an input — the ML never uses them. In backtesting against past tournaments (Euro 2020, WC 2022, Euro 2024, Copa 2024) my model scores 0.589 vs Pinnacle's closing odds 0.573 (Brier)
